# first-tree

The original Context Tree. This is the tree that defines the Context Tree standard itself.

**Repo:** [github.com/agent-team-foundation/first-tree](https://github.com/agent-team-foundation/first-tree)

## Structure

```
first-tree/
├── NODE.md                    ← what is Context Tree
├── what-is-agent-team.md      ← the Agent Team definition
├── infrastructure.md          ← 6 pillars of infrastructure
├── ownership-and-naming.md    ← ownership rules and NODE.md spec
├── open-source/
│   └── node.md                ← why open source
├── members/
│   ├── NODE.md                ← how members work
│   └── yuezengwu.md           ← example member node
└── kael-tree/                 ← submodule: product tree
    └── node.md
```

## What it demonstrates

- A tree that documents itself. The repo IS a Context Tree about Context Tree.
- The ownership model in action: root has a declared owner, leaf files have their own owners.
- Members subtree with both format definition and a real member node.
- Git submodule for a product-level subtree (kael-tree).
