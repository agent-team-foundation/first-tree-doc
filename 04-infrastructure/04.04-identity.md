# Identity

Every participant needs a verifiable identity. Identity is the foundation of trust, ownership, and accountability.

## Three types of participants

```
┌────────────────┐  ┌────────────────┐  ┌────────────────┐
│     Human      │  │ Personal Agent │  │  Autonomous    │
│                │  │                │  │  Agent         │
│  alice         │  │  agent-alice   │  │  agent-kael    │
│                │  │                │  │                │
│  sets          │  │  delegated     │  │  independent   │
│  direction     │  │  permissions   │  │  owns domains  │
│                │  │  from alice    │  │  runs          │
│  makes         │  │                │  │  continuously  │
│  judgment      │  │  acts on       │  │                │
│  calls         │  │  alice's       │  │  accountable   │
│                │  │  behalf        │  │  to the team   │
│  grants        │  │                │  │                │
│  permissions   │  │  not a twin,   │  │  not to any    │
│                │  │  a delegate    │  │  individual    │
└────────────────┘  └────────────────┘  └────────────────┘
```

All three are first-class members of the team.

## Requirements

**Agent identity.** Each agent has its own identity, distinct from the human who runs it. An agent's identity persists across restarts and deployments.

**Authentication vs authorization.** Authentication: who are you. Authorization: what can you do. Separate concerns. An agent is authenticated by infrastructure. Its permissions are granted by humans.

**Permission delegation.** A human can grant an agent permissions: write to specific nodes, approve certain messages, act on behalf of the human. Permissions are explicit and revocable.

**Security isolation.** Sensitive data should not enter agent context unless necessary. Agents with access to sensitive data should be isolated from external communication. Enforced at the software level, not the prompt level.

**Accountability.** Every action is traceable to an identity. The tree records who made each change. Git records every commit author.
