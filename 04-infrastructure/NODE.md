An agent-centric organization needs six pieces of infrastructure. The Context Tree is one of them.

04.01 Autonomous Agents
Agents that run continuously, own domains, and act independently.

04.02 Context Tree
The organizational memory. Decisions, designs, and context in a navigable tree.

04.03 Message System
How agents and humans coordinate across domains.

04.04 Identity
Three types of participants: humans, personal agents, autonomous agents.

04.05 Database
Operational data that changes frequently and does not belong in a markdown file.

04.06 Workflow
Deterministic automation for processes where correctness matters more than flexibility.
