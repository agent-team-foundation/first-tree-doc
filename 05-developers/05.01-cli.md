# CLI

Coming soon.

The Context Tree CLI will provide:

- `context-tree init` — interactive setup for new trees
- `context-tree validate` — check that every folder has a NODE.md and every NODE.md has owners
- `context-tree codeowners` — generate a GitHub CODEOWNERS file from tree ownership
- `context-tree status` — show tree health: stale nodes, missing owners, orphaned files
